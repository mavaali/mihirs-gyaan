# Contributing

This is a personal corpus of agent skills. Contributions are welcome when they
add portable judgment — a skill that encodes how to do a specific job well,
written so any agent can load it and follow it.

Before opening a PR, skim the [conventions in the README](./README.md#conventions):
one kebab-case folder per skill, a `SKILL.md` as the only required file, and no
hardcoded personal paths or org-internal tooling.

## Vendor-promotion PRs

PRs that primarily exist to link to a single commercial service — typically via
an automated submission referencing a vendor's docs, sometimes by an author
papering many skill collections at once — will be closed without further review.
Skills are expected to encode portable judgment and procedural know-how, not act
as a thin referral to an upstream vendor.
